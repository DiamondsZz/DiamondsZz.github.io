offset.xxx与style.xxx的区别

  • style.left只能获取行内的，而offsetLeft则可以获取到所有的；
  • offsetLeft 可以返回没有定位盒子距离左侧的位置；而style.left不可以，其只能返回有定位盒子的left;
  • offsetLeft 返回的是数字，而 style.left 返回的是字符串，除了数字外还带有单位：px;
  • 注意：可以用parseInt进行转化;比如：styleLeft='300px' —> parseInt(styleLft) —> 300
  • offsetLeft是只读的，而style.left是可读写；
  • 如果没有给当前元素指定过top样式，则 style.top返回的是空字符串。

子元素的margin-top影响父元素

所有毗邻的两个或更多盒元素的margin将会合并为一个margin共享之。
毗邻的定义为：同级或者嵌套的盒元素，并且它们之间没有非空内容、padding或Border分隔。

解决办法：
1.给父元素加上padding/border/overflow:hidden，
2.或者给父元素/子元素设置为float/position:absolute.（浮动元素和绝对定位元素不参与margin折叠）

css盒子

  1. 如果需要控制嵌套关系盒子之间的距离，应首先考虑padding（父子之间）,其次再考虑margin。（兄弟之间）
  2. margin在垂直方向会合并（依据较大的）。
  3. 浮动元素的位置根据元素浮动之前在标准文档流中的位置所决定。
  4. margin/padding 百分比根据父元素宽度来计算。translate依据自身。

定位
绝对定位依据浏览器窗口首屏宽高而不是整个浏览器窗口的宽高。
相对定位不脱离文档流，占据空间，不利于布局。

inline-block
div设置inline-block不能在排一行
原因：可能是因为你标签间有间隙，空格一起渲染，解决办法可以是把两个div标签间的空隙注释掉，应该就可以了。

    2019-3-15

    em/rem/px

  • em:值并不是固定的,继承父级元素的字体大小(相对于当前对象内文本的字体尺寸。如当前对行内文本的字体尺寸未被人为设置，则相对于浏览器的默认字体尺寸。任意浏览器的默认字体高都是16px。所有未经调整的浏览器都符合: 1em=16px,在css中的body选择器中声明Font-size=62.5%，这就使em值变为 16px*62.5%=10px, 这样12px=1.2em, 10px=1em)在父级元素中声明了字体大小为1.2em，那么在子元素的字体大小时就只能是1em，而不是1.2em, 因为此em非彼em，它因继承父级元素的字体高而变为了1em=12px。
  • rem:相对HTML根元素
  • px:相对于显示器屏幕分辨率

柯里化

  • 所谓‘柯里化’，就是把一个多参数的函数，转化成单参数并且返回接受余下的参数且返回结果的新函数。（我的天，好绕啊！！！）
//柯里化之前
function add(x,y)
{
    return x+y;

//柯里化之后

{
    return function(y){
        return x+y;

    寄生组合式继承

//构建Person类
function Person(name, age) {
    this.name = name;
    this.age = age;
}
//Person类的原型对象 添加run方法
Person.prototype.run = function () {
    console.log(this.name + '人跑步');
};

//构建Student类
function Student(name, age, id) {

    Person.call(this, name, age); //实现Person构造函数的属性和方法继承
    this.name = name;
    this.age = age;
    this.id = id;
    //Student类的run方法
    this.run = function () {
        console.log(this.name + '学生跑步');
    }
}
//构建一个临时的构造函数（第三者）为了避免Student类污染Person类
function Temp() {}
//实现Temp对Person继承
①Temp.prototype = Person.prototype;
//作为Student类的原型对象
var stu = new Temp();
//实现Student对Person继承(间接)
Student.prototype = stu;
//此时 Student.prototype.constructor===stu.constructor,stu是个实例化对象，stu.constructor===stu.__proto__.constructor,stu.__proto__===stu.constructor.prototype,实例化对象stu的构造函数是Temp,由①可得stu.constructor===Person.prototype.constructor。所以此时Student.prototype.constructor不是自己的构造函数，应该设置为自己的构造函数Student。
Student.prototype.constructor=Student;

once函数

function once(fn) {
    var flag = true;

    return function () {    //fn=once(fn) 利用闭包保证函数不被销毁,实现一次执行
        if (flag) {
            flag = false;
            fn();
        }
    }
}

函数节流与函数防抖

函数节流与防抖图示：
图片链接:https://blog.coding.net/blog/the-difference-between-throttle-and-debounce-in-underscorejs

代码借鉴于underscore.js,感觉源码值得学习一下

  • 函数节流
function throttle(fn, wait,options) {

    var timeout, result;
    var previous = 0;
    if (!options) options = {};

    var later = function () {
        previous = options.leading === false ? 0 :  Date.now();
        timeout = null;
        result = fn.apply(null);
    };

    var throttled = function () {

        var now = Date.now();


        if (!previous && options.leading === false) previous = now;

        //计算剩余时间
        var remaining = wait - (now - previous);

        if (remaining <= 0 || remaining > wait) {   //设置options中leading为false时 remaining=wait  不执行函数
            if (timeout) {
                clearTimeout(timeout);
                timeout = null;
            }
            previous = now;
            result = fn.apply(null);
        } else if (!timeout && options.trailing !== false) {
            timeout = setTimeout(later, remaining);
        }
        return result;
    };
    return throttled;
}
  • 函数防抖
function debounce(func, wait, immediate) {
    var timeout, result;

    var later = function () {
        timeout = null;
        result = func.apply(null);
    };

    var debounced =function () {
        if (timeout) clearTimeout(timeout);
        if (immediate) {
            var callNow = !timeout;
            timeout = setTimeout(later, wait);
            if (callNow) result = func.apply(null);
        } else {
            timeout = setTimeout(later, wait);
        }

        return result;
    };

    return debounced;
}
