offset.xxx与style.xxx的区别

  • style.left只能获取行内的，而offsetLeft则可以获取到所有的；
  • offsetLeft 可以返回没有定位盒子距离左侧的位置；而style.left不可以，其只能返回有定位盒子的left;
  • offsetLeft 返回的是数字，而 style.left 返回的是字符串，除了数字外还带有单位：px;
  • 注意：可以用parseInt进行转化;比如：styleLeft='300px' —> parseInt(styleLft) —> 300
  • offsetLeft是只读的，而style.left是可读写；
  • 如果没有给当前元素指定过top样式，则 style.top返回的是空字符串。

子元素的margin-top影响父元素

所有毗邻的两个或更多盒元素的margin将会合并为一个margin共享之。
毗邻的定义为：同级或者嵌套的盒元素，并且它们之间没有非空内容、padding或Border分隔。

解决办法：
1.给父元素加上padding/border/overflow:hidden(IE要添加zoom:1)，
2.或者给父元素/子元素设置为float/position:absolute.（浮动元素和绝对定位元素不参与margin折叠）

文字溢出

单行溢出

width: xxx;
overflow: hidden;
white-space: nowrap;
text-overflow: ellipsis;

text-overflow:ellipsis;要达到的效果是：文字超出容器宽度时，文字被隐藏的文字用省略号代替。所以该属性只能用于块状元素或行内块元素中，对行内元素是不起作用的。

多行溢出（仅适用于谷歌浏览器，可通过js来实现）

width: xxx;
overflow: hidden;  
text-overflow: ellipsis;  
display: -webkit-box;  
-webkit-line-clamp:n; 第n行显示省略号
-webkit-box-orient: vertical;

原型链

  1. 原型链的弊端是不支持多重继承。记住，原型链会用另一类型的对象重写类的 prototype 属性。
  2. 要确保构造函数没有任何参数。

call 和apply

call() 方法

  1. 它的第一个参数用作 this 的对象。其他参数都直接传递给函数自身。

apply() 方法

  1. apply() 方法有两个参数，用作 this 的对象和要传递给函数的参数的数组。
  2. 只有超类中的参数顺序与子类中的参数顺序完全一致时才可以传递参数对象。如果不是，就必须创建一个单独的数组，按照正确的顺序放置参数。

css盒子

  1. 如果需要控制嵌套关系盒子之间的距离，应首先考虑padding（父子之间）,其次再考虑margin。（兄弟之间）
  2. margin在垂直方向会合并（依据较大的）。
  3. 浮动元素的位置根据元素浮动之前在标准文档流中的位置所决定。

清除浮动的六种方式

  1. 给上面一个盒子设置高度
  2. 使用clear属性
  3. 隔墙法：
    内墙法：给上一个盒子的所有子元素的后面添加一个块级元素，设置clear属性（第一个盒子可以使用margin-bottom属性，第二个盒子可以使用margin-top属性）
  4. 外墙法：给两个盒子中间添加一个块级元素，设置clear属性（第一个盒子不能使用margin-bottom属性，第二个盒子可以使用margin-top属性，可以设置添加盒子的高度）
  5. 使用伪元素选择器：标签名：：after/before{content:" "; height:0; clear:both; visibility:hidden; } 兼容ie6 通过属性(*zoom:1;)
  6. 第一个盒子设置overflow:hidden;属性 兼容ie6 通过属性(*zoom:1;)
    设置了overflow:hidden;属性，内盒子设置margin-top属性父元素不会被顶下来（不用再设置父元素的边框border属性）

定位
绝对定位依据浏览器窗口首屏宽高而不是整个浏览器窗口的宽高。
相对定位不脱离文档流，占据空间，不利于布局。

inline-block
div设置inline-block不能在排一行
原因：可能是因为你标签间有间隙，空格一起渲染，解决办法可以是把两个div标签间的空隙注释掉，应该就可以了。

js

1.“！”是逻辑与运算，并且可以与任何变量进行逻辑与将其转化为布尔值，“!!”则是逻辑与的取反运算，尤其后者在判断类型时代码简洁高效，省去了多次判断null、undefined和空字符串的冗余代码。

nth-child选择器

1.nth-child(n+2):nth-child(odd):nth-child(-n+9)
使用 nth-child(n+2):nth-child(odd):nth-child(-n+9) 我们将会选中的子元素是从第2位到第9位，并且只包含奇数位。
